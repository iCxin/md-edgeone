# 更新日志

## [2.1.0-cxin] - 2024-12-07

### 🎉 魔改版特性

- **项目结构优化**
  - 移除 `apps/vscode`、`apps/utools` 和不必要的 `packages` 目录
  - 精简项目架构，专注核心功能
  - 适配静态Pages服务部署

- **功能精简**
  - 移除所有AI功能模块和相关组件
  - 删除AI配置、图像生成等高级功能
  - 专注于纯粹的Markdown编辑体验

- **界面优化**
  - 删除帮助文档和推广内容
  - 移除赞助商信息和相关链接
  - 界面更加简洁专注

- **公式显示改进**
  - 统一所有数学公式的字体大小
  - 为超长公式添加横向滚动条支持
  - 改善公式完整显示效果

- **部署方案优化**
  - 新建根目录 `vite.config.ts` 配置文件
  - 构建输出到 `dist/` 目录
  - 复制 `index.html` 到根目录

### 📝 文档更新

- 更新项目标题为 "微信Markdown编辑器 | www.cxin.net魔改版"
- 添加原创作者和二改作者信息标注
- 创建新的README文档，突出二次开发特性
- 添加部署指南文档

### 🔧 技术改进

- 更新 package.json 项目信息
- 简化构建脚本和依赖管理
- 移除不必要的工具和配置文件

### 📦 包管理

- 移除AI相关依赖包
- 精简开发工具链
- 保持核心功能完整
