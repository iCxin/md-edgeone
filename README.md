## 📝 项目介绍

**Markdown 文档自动即时渲染为微信图文**，让你不再为微信内容排版而发愁！只要你会基本的 Markdown 语法，就能做出一篇样式简洁而又美观大方的微信图文。

本项目是基于 @doocs 的优秀开源项目进行二次开发的魔改版本，主要优化：
- 移除了所有AI相关功能，专注于纯粹的Markdown编辑体验
- 优化了项目结构，适配Pages服务部署
- 改进了数学公式显示效果，统一字体大小并支持横向滚动
- 删除了推广内容和帮助文档，界面更简洁
